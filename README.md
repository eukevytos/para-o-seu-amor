# ❤️ Nosso Amor em Cada Detalhe

Um projeto web minimalista e interativo criado para celebrar momentos especiais, histórias e memórias de um relacionamento. O site combina um design elegante com funcionalidades interativas como um quiz personalizado.

## 🚀 Demonstração
O site é composto por três seções principais:
1. **Landing Page:** Introdução visual baseada em uma estética clean e romântica.
2. **Razões do Amor:** Um grid interativo de cards detalhando sentimentos.
3. **Quiz Interativo:** Um jogo de múltipla escolha para testar os conhecimentos do parceiro(a) sobre a história do casal.

## 🛠️ Tecnologias Utilizadas
Este projeto foi desenvolvido utilizando tecnologias web fundamentais (Vanilla Stack), focando em performance e responsividade:
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização personalizada, Flexbox, CSS Grid e animações de transição.
* **JavaScript (ES6):** Lógica do Quiz, manipulação de DOM e efeitos de rolagem suave.
* **Google Fonts:** Integração das fontes *Bodoni Moda*, *Montserrat* e *Great Vibes*.
